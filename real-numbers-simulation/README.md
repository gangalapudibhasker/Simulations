# Real Numbers Simulation - CBSE 10th Class Mathematics

An interactive React application that helps students understand **Euclid's Division Algorithm** and the **Fundamental Theorem of Arithmetic** through visual simulations, practice problems, and quizzes.

## Features

### 📚 Theory Section
- **Euclid's Division Algorithm**: Complete explanation with definitions, formulas, and examples
- **Fundamental Theorem of Arithmetic**: Detailed theory with prime factorization examples
- **Finding HCF**: Step-by-step guide using Euclid's algorithm

### 🎯 Algorithm Visualizer
- Interactive visualization of Euclid's Division Algorithm
- Step-by-step animation showing the division process
- Visual representation of numbers and remainders
- Random example generator
- Real-time calculation of HCF

### 💡 Practice Section
- 10 randomly generated practice problems
- Instant feedback on answers
- Detailed solutions with step-by-step working
- Progress tracking
- Optional timer for speed practice

### 📝 Quiz Section
- 10 multiple-choice questions
- 5-minute time limit
- Instant feedback with explanations
- Score tracking and performance analysis
- Randomized question order

## Topics Covered

1. **Euclid's Division Algorithm**: Understanding the relationship between dividend, divisor, quotient, and remainder
2. **Finding HCF**: Using the algorithm to find the Highest Common Factor
3. **Fundamental Theorem of Arithmetic**: Unique prime factorization of integers
4. **Properties of HCF**: Commutative, associative, and distributive properties

## Installation

```bash
# Navigate to the project directory
cd real-numbers-simulation

# Install dependencies
npm install

# Start the development server
npm start
```

The application will open in your default browser at `http://localhost:3000`

## Usage

1. **Theory**: Read through the concepts and examples
2. **Visualizer**: Enter two numbers to see the algorithm in action
3. **Practice**: Solve problems to test your understanding
4. **Quiz**: Take a timed quiz to assess your knowledge

## Technologies Used

- **React 18**: Frontend framework
- **Framer Motion**: Smooth animations and transitions
- **Styled Components**: CSS-in-JS for styling
- **Responsive Design**: Works on all screen sizes

## CBSE Curriculum Alignment

This simulation covers the first topic of **Chapter 1: Real Numbers** from the CBSE Class 10 Mathematics syllabus:

- **1.1 Introduction** to real numbers
- **1.2 Euclid's Division Algorithm**
- **1.3 The Fundamental Theorem of Arithmetic**

## Learning Objectives

By using this simulation, students will be able to:
- Understand and apply Euclid's Division Algorithm
- Find the HCF of two numbers using the algorithm
- Understand the concept of prime factorization
- Apply the Fundamental Theorem of Arithmetic
- Solve problems related to real numbers
